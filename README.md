## About
```
此项目是 vue2 + element-ui 构建的后台管理系统，使用mockjs模拟后端的数据，并使用echarts进行图表的展示。
```

### 项目运行
```
---- yarn install
---- yarn serve
打包：
---- yarn build
```

### 技术栈
```
vue2 + vue-router + vuex + element-ui + axios + mockjs + echarts
```
